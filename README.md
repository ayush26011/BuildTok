# BuildTok

BuildTok is a full-stack social media platform where developers, designers, students, and creators can showcase their projects through short-form vertical videos.

It is designed like a mix of Instagram Reels and LinkedIn for builders — users can upload project videos, discover creators, like, comment, follow, message each other, and share projects in real-time chats.

## Live Demo

Frontend: https://build-tok.vercel.app
Backend: https://buildtok.onrender.com

## Project Overview

BuildTok helps creators show what they have built instead of only describing it. Users can publish project reels with descriptions, tech stacks, GitHub links, and live demo links.

The platform includes a complete authentication system, project feed, creator profiles, social interactions, real-time messaging, media uploads, and a functional settings system.
